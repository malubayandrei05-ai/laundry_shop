Laundry Shop Management System - Enhanced Glassmorphism Version

Built with PHP and MySQL.

Features:
- Login and sign up
- Plain-text password storage, no hashing
- Modern glassmorphism UI
- Lightning visual effects
- Dashboard summary cards
- Customer management
- Service management
- Laundry product management
- Order management
- Payment tracking
- Sales reports

Setup:
1. Copy the folder to htdocs if using XAMPP.
2. Open phpMyAdmin.
3. Create/import using sql/laundry_shop.sql.
4. Check database credentials in config.php.
5. Open http://localhost/laundry_enhanced/login.php or your folder name.
